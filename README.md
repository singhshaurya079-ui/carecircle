# 🦘 CareCircle - Modern Caregiver Companion App

CareCircle is a comprehensive, mobile-first web application designed specifically for caregivers and households managing one or multiple patients. It moves away from cluttered single-page dashboards into a sleek, fast, and organized multi-view interface optimized for both desktop and mobile viewports.

## 🚀 Key Features

- **📱 App-Style Multi-View Navigation:** Smoothly switch between isolated panels for Overview, Tasks, Appointments, Notes, Medicines, Contacts, and Surgical Procedures without reloading the page.
- **⏱️ Real-Time Countdown Alerts:** A live, ticking interval engine built into the dashboard that calculates and displays exact countdowns (e.g., "Surgery in: 2h 14m 05s") for critical events.
- **🔊 Audible Medication Alarms:** An active background clock listener that cross-checks your local device time and triggers a native audible/text-to-speech announcement ("*Time for medicine: [Medicine Name]*") at the exact minute set by the caregiver.
- **🗂️ Categorized Contacts Directory:** Visually separates your support network into "Family & Caregivers" and "Medical Professionals" (Doctors, Nurses, Surgeons) with dedicated input flows capturing full names, roles, and country codes.
- **🔍 Instant Keyword Filter Search:** A dedicated search bar embedded directly into dashboard views to instantly filter items and contacts in real time as you type.
- **💾 Auto-Sync Local Storage:** Automatically stringifies and backs up all profile tracking arrays onto device memory so no records are lost during mobile browser refreshes.

## 🛠️ Built With

- **HTML5** - Semantic structure and native dialog layout modals.
- **CSS3** - Modern grid/flexbox layouts, responsive app styling breakpoints, and interactive hover mechanics.
- **JavaScript (ES6+)** - Ticking countdown engines, background event routing, local data state management, and Web Audio API integration.
